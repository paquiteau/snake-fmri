class NoiseHandler(snr: float = 0)#

Bases: AbstractHandler

Add noise to the data.

Parameters:
snr

The target SNR The SNR is defined as max(signal) / std(noise)

verbose

verbose flag. If True, a callback function is setup to return the computed snr.

Attributes:
callbacks

Return the list of callbacks run after the handling.
